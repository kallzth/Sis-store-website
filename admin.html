🔑 Reset Admin Password

Since this is a private admin panel, your password is stored locally in your browser. To reset it:

1.Open browser Console (F12 → Console tab)
2.Type: localStorage.setItem('sisay_admin_pass','yournewpassword')
3.Press Enter → then try logging in with the new password

Or contact your developer to reset it directly.
